Conocimientos Adquiridos

  • Javascript

    Lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, ​ basado en prototipos, imperativo, débilmente tipado y dinámico.

  • Jquery

    Libreria de javascript que permite simplificar la manera de interactuar con los documentos HTML, manipular el árbol DOM, manejar eventos, desarrollar animaciones y agregar interacción con la técnica AJAX a páginas web.

  • MySql

    Sistema de gestión de bases de datos relacionales de código abierto con un modelo cliente-servidor.

  • Wordpress

    Sistema de gestión de contenidos lanzado el 27 de mayo de 2003, enfocado a la creación de cualquier tipo de página web.

  • Bootstrap

    Libreria multiplataforma o conjunto de herramientas de código abierto para diseño de sitios y aplicaciones.

  • Css

    Lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcad.

  • Html

    Lenguaje de marcación que sirve para definir el contenido de las páginas web. Se compone en base a etiquetas, también llamadas marcas o tags.

  • PHP

    Lenguaje de programación de uso general que se adapta especialmente al desarrollo web en el backend.
